# 88 Days WA Tool

A searchable database of 88-day eligible jobs in Western Australia for Working Holiday Visa holders.

## Project Phases
- Phase 1: Data foundation (postcode registry, job categories)
- Phase 2: Web scraping & verification
- Phase 3: API backend
- Phase 4: React frontend
- Phase 5: Deployment

## Tech Stack
- Backend: Node.js + Express
- Database: SQLite
- Frontend: React + Tailwind CSS
- Hosting: Railway.app or Render.com
